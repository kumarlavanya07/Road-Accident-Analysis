# Road Accident Dashboard – Project Description

## Overview
This project is an **interactive Road Accident Analytics Dashboard** built in Microsoft Excel,
designed to visualize and analyze road accident casualty data across multiple dimensions including
time, vehicle type, road type, surface condition, and location.

---

## Objective
To provide traffic safety analysts, government agencies, and policymakers with a clear,
data-driven view of road accident trends and casualty breakdowns — enabling informed decisions
to reduce road fatalities and injuries.

---

## Key Metrics Tracked

| KPI | Value |
|---|---|
| Total Casualties | 4,17,883 |
| Fatal Casualties | 7,135 (1.7%) |
| Serious Casualties | 59,312 (14.2%) |
| Slight Casualties | 3,51,436 (84.1%) |
| Car Casualties | 3,33,485 (79.8%) |

---

## Dashboard Components

- **KPI Cards** — Instant snapshot of fatal, serious, slight, and vehicle-specific casualties
- **Monthly Trend Line Chart** — Year-over-year comparison (2021 vs 2022) of casualty volumes
- **Vehicle Type Breakdown** — Casualties segmented by cars, trucks, buses, bikes, tractors, and others
- **Road Type Analysis** — Bar chart comparing single carriageway, dual carriageway, roundabout, etc.
- **Road Surface Conditions** — Dry, wet, and snow/ice surface impact on accidents
- **Location Analysis** — Rural vs Urban casualty split (donut chart)
- **Light Condition Analysis** — Daylight vs Night accident distribution
- **Interactive Filter Panel** — Dynamic slicers for Accident Date (2021–2023) and Urban/Rural toggle

---

## Tools & Techniques Used

- **Microsoft Excel** — PivotTables, PivotCharts, Slicers, Donut/Bar/Line Charts
- **Dashboard Design** — Dark-themed UI with color-coded KPIs for quick visual scanning
- **Data Sheet Architecture** — Separate tabs for raw DATA, KPI, Monthly, RoadType, and Sheet views

---

## Target Audience
Traffic authorities, urban planners, road safety researchers, and data analytics portfolio reviewers.
